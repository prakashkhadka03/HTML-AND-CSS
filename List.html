HTML Lists

HTML offers different types of lists to organize content:

Ordered List

An ordered list is a list that is numbered:

  1. First item
  2. Second item
  3. Third item

An unordered HTML list

An unordered list is a list that uses bullet points:

  • coffee
  • tea
  • milk

A Description List

>coffee
- black hot drink
Milk
- white cold drink
